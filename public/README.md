# Public Assets Directory

## Required Files

### NOVO_Primary.png
Place the NOVO logo file (`NOVO_Primary.png`) in this directory.

The logo is used on the onboarding welcome screen and should be:
- File name: `NOVO_Primary.png` (exactly as shown)
- Recommended dimensions: 120-150px width
- Format: PNG with transparent background preferred
- Location: `/public/NOVO_Primary.png`

The file will be accessible at `/NOVO_Primary.png` in the application.
